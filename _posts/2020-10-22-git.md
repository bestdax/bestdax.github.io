# 前进和回退

```
git log # 列出所有的版本，HEAD表示当前版本
git log --pretty=oneline # 列出所有版本的简化信息
git reset HEAD^ # 回退到上个版本。 HEAD^^回退到上两个版本，HEAD~100回退到前100个版本。或者可以提供版本号，版本号不一定要写全。
git reflog # 可以看到版本操作的记录，里面可以查到对应版本的编号
```
